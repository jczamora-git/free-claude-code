How to Use Claude Code with Agent Router in VS Code for Free

This guide will show you how to install Anthropic's official Claude Code tool on Windows and connect it through Agent Router so you can use your promotional API credits.

Reminder: Never share your API key publicly. Do not upload it to GitHub, screenshots, or public posts.
Step 1

Claim Your Free API Key

Required Registration Link
Important: Register using the link below.

You must create your Agent Router account through this specific promo link to qualify for the $150 promotional credit. Registering directly on the website without using this link may prevent the promotional credit from being applied to your account.

Register through the required promo link
  1. Click the registration button above. Do not manually search for or open the Agent Router registration page separately.
  2. Sign in or register using your GitHub account.
  3. After registration, confirm that the $150 promotional credit has been applied to your account.
  4. Navigate to the API Token page.
  5. Create and copy your API key. It will look something like: sk-your-unique-key

Save your API key securely because you will need it later in Step 4. Never share it publicly or include it in screenshots.

Step 2

Install Node.js

Claude Code requires Node.js to run on your computer.

  1. Visit the official Node.js website.
  2. Download the LTS version.
  3. Run the installer.
  4. Keep the default settings checked.
  5. Finish the installation.
Step 3

Install Claude Code in VS Code

Windows may block some developer tools from installing through PowerShell. Run this command first to allow script execution for your user account. 

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

After that, install Claude Code globally using npm: 

npm install -g @anthropic-ai/claude-code

Wait until the installation is fully completed before moving to the next step.

Step 4

Configure the Proxy Settings

Now, open the hidden Claude settings file by running this command in the VS Code terminal: 

notepad $HOME\.claude\settings.json

If Notepad says the file or folder does not exist, run this first: 

claude

Then press Ctrl + C to exit, and try opening the settings file again.

Once Notepad opens, delete everything inside and paste this configuration: 

{
  "env": {
    "ANTHROPIC_BASE_URL": "https://agentrouter.org/",
    "ANTHROPIC_AUTH_TOKEN": "sk-your-agentrouter-key",
    "ANTHROPIC_MODEL": "claude-opus-4-6"
  },
  "alwaysThinkingEnabled": true,
  "model": "claude-opus-4-6",
  "effortLevel": "high"
}

Replace sk-your-agentrouter-key with the actual API key you copied earlier. Make sure your key stays inside the quotation marks.

Save the file using Ctrl + S, then close Notepad.

Step 5

Launch and Test Claude Code

Go back to your VS Code terminal and start Claude Code using this command: 

claude --model claude-opus-4-6

If Claude Code asks you to set up terminal settings, choose: 

1. Yes, use recommended settings

Then type: 

hello
If Claude replies to your greeting, your setup is working. You can now use Claude Code in VS Code through Agent Router using your available API credits.
